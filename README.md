# Lots Of Interesting Stuff Knowledge Graph

Building a knowledge graph of taxonomic data.



## Setup

```
composer init
echo "vendor/" > .gitignore
composer update
```

## Technology

--|--
Technology | Provider
Document database | CouchDB
Triple store | Blazegraph
Templating | EJS

## Named graphs

--|--
Source | Graph URI
IPNI | https://www.ipni.org
ORCID | https://orcid.org
